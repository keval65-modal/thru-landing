USER MIGRATION - BROWSER CONSOLE COMMAND
=========================================

1. Go to your live app: https://thru-user-app29082025-master-l02sglt1u-keval65-modals-projects.vercel.app

2. Open Developer Console (F12 or right-click ‚Üí Inspect ‚Üí Console)

3. Copy and paste this entire command:

```javascript
(async function migrateUsers() {
  console.log('üöÄ Starting User Migration...');
  try {
    const usersRef = firebase.firestore().collection('users');
    const usersSnapshot = await usersRef.get();
    console.log(`Found ${usersSnapshot.size} users to check`);
    
    let updatedCount = 0;
    for (const userDoc of usersSnapshot.docs) {
      const userData = userDoc.data();
      if (!userData.displayName) {
        const displayName = `${userData.profileData.name},${userData.profileData.phoneNumber}`;
        await userDoc.ref.update({ displayName });
        console.log(`‚úÖ Updated: ${displayName}`);
        updatedCount++;
      }
    }
    console.log(`üéâ Migration complete! Updated ${updatedCount} users.`);
  } catch (error) {
    console.error('‚ùå Migration failed:', error);
  }
})();
```

4. Press Enter to run the migration

5. Check the console output for results
